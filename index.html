Dynamize.js

Small plain vanilla javascript goodies for enhancing user interaction

View project on GitHub

When enhancing a website for eye-candy and user interaction, there is nothing better than finding a pure CSS + HTML solution. When such a solution is not available, the developer needs to resort to javascript. That was the origin of Dynamize.js: a loose collection of small, independient, plain vanilla javascript files to do one specific task and designed to avoid messing with the existing site layout.

In other words, if you need to add a particular functionality covered by one Dynamize tool, just grab the file, drop it in your existing website project and there is a good chance that everything is going to work as expected regardless of the nature of the project, even when external javascript frameworks are used.
