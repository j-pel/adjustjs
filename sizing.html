sizing.js

Sizing things around in a website is helpful for windowing activities, simulating a desktop working environment. And there are many similar use cases for a web application. The DOM had already everything needed in order to provide that interaction. No need to load a framework for that. Just provide a function to store the object's new size in case persistance is required.

In order to have a sizable object, there is only need to include a script tag at the end of the <body> and add a class for the object that is going to be sized. All css is preserved and there is a small specific markup required: the objects have to be absolutely positioned.
