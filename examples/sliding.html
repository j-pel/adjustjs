Sliding.js

This tool is useful for several scenarios. From slide presentation layouts and image carousels to touch friendly pagination systems.

It is simple to use. Just identify the items that should be made slidable. Then append each slide as an object, including a function to draw the slide. Each slide could have a different draw function and different information on it.

Use case 1: Slides

    <div class="slidable" style="border: 1px solid black;">
    </div>
    <script type="text/javascript">

      sliding.appendSlide({title: "Slide one", draw: paintSlide});
      sliding.appendSlide({title: "Slide two", draw: paintSlide});
      sliding.appendSlide({title: "Slide three", color: 'yellow', draw: paintColorSlide});

      function paintSlide(obj, info){
        obj.innerHTML = info.title;
      }

      function paintColorSlide(obj, info){
        obj.innerHTML = info.title;
        obj.style.backgroundColor = info.color;
      }

    </script>
