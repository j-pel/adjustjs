Sliding.js

This tool is useful for several scenarios. From slide presentation layouts and image carousels to touch friendly pagination systems.

It is simple to use. Just identify the items that should be made slidable. They are going to work as viewports for the sildes. Each slide is appended as a javascript object. Any data could be passed inside that object, including a function to draw the slide. Each slide could have a different draw function and different information on it.

NOTE: The draw function is free to modify the viewport. This could have undesired side effects because those changes persist between slide changes.
