Filtering.js

The idea with this widget is to use table headers as text boxes for filtering table contents.

Any table could be applied to the filtering object and the rest is handled automatically.

The filter is equally applied to tables with multiple bodies.

Filtering could work together with scrolling for tables. Just assign filtrable and/or scrollable classes.
