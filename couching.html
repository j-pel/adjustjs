couching.js

Couching is a very small wrapper around Apache CouchDB REST API. All functions are implemented as simple AJAX callbacks. It is useful, among other things, for persist the user interactions on any Dynamize.js tool.

CouchDB is reliable, easy to use, flexible and fast. There are also several database service providers that let you run CouchDB instances for free. That's a lot of power to drive your own cloud.

The database could be located on any server such as IrisCouch, Cloudant or on a local running instance of CouchDB.

Couch databases are stores for freeform JSON documents. Just create a database and start feeding documents into it. To retrieve a particular document, use the document id. To retrieve several related documents, a view should be created as a javascript map function iterating through all documents in the database.

In order to use it for persisting the kind of interactions derived from Dynamize, just create a database and store a document for each UI element. Then update the document each time the element changes, by means of the store function on each tool.
