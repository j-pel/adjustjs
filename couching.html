couching.js

Couching is a very small wrapper around Apache CouchDB REST API and all compatible variations, forks and clones. All functions are implemented as simple AJAX callbacks. It is useful, among other things, for persist the user interactions on any Dynamize.js tool.

CouchDB is wonderful, reliable, easy, flexible and fast. There are also several database service providers that let you run CouchDB instances for free. That's a lot of power for having your own cloud.

Couch databases are stores for freeform JSON documents. In order to use it, just create a store function to capture user interaction and then create a JSON document to put it in an existing database. The database could be located on any server such as IrisCouch, Cloudant or a local instance of CouchDB.
