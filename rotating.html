rotating.js

Rotating things around in a website is not a very common task. But for a web application, it is very handy and open a lot of possibilities. The DOM had already everything needed in order to provide that interaction. No need to load a framework for that. Just provide a function to store the object's transformation in case persistance is required.

In order to have a rotable object, there is only need to include a script tag at the end of the <body> and add a class for the object that is going to rotate. All css is preserved and there is a small specific markup required: the objects have to be absolutely positioned.

Use case:

<body>
	<div id="f1" class="small-box" style="top: 0px; left: 175px;">I'm fixed</div>
	<div id="m1" class="small-box rotable" style="top: 0px; left: 260px;">Move me</div>
	<div id="m2" class="small-box rotable" style="top: 0px; left: 345px; background-color:#cce;">Move me too!</div>

			(...)

	<script type="text/javascript" src="javascripts/rotating.js"></script>
	<script>
		var store_changes = function(changes) {
			console.log("storing",changes);
			return 0;
		}
		rotating.init(store_changes);
	</script>
</body>
